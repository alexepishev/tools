## Build

Set proper GOROOT and PATH variables
```sh
export GOROOT="$HOME/go/root/1.17.4"
export PATH="$PATH:$GOROOT/bin"
```

Clone repository
```sh
git clone https://github.com/devopsext/tools.git
cd tools/
go build
```
